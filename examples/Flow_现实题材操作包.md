# Flow 现实题材操作包

## 适用场景

- 用户明确说会在 `Google Flow` 中执行
- 题材涉及 `现实人物`、`旧职业`、`地方记忆`、`时代变迁`
- 目标不是一句 prompt，而是完整制作流程

## 推荐调用方式

```text
使用 $gen-video，把这个现实题材故事改写成适合 Google Flow 的完整操作包。
要求先拆剧情结构，再输出素材图提示词、镜头表、关键帧回灌点和 Scenebuilder 拼接顺序。
```

## 用户输入示例

```text
使用 $gen-video，把这个故事改写成适合 Google Flow 的完整操作包：

一位快退休的老电影院放映员守着最后一台 16 毫米放映机，小影院即将停业。一个做社区影像志愿服务的高中生，用 AI 修复旧海报和残损胶片，在拆除前帮他完成最后一场露天放映。整体参考现实题材催泪短片，先抑后扬，高潮不靠大特效，而靠那束重新亮起来的银幕光。
```

## 预期输出

- 项目参数建议
- 模型栈建议
- 故事 beat 结构
- 角色 / 场景 / 道具锚点
- Nano Banana 素材图清单
- 按镜头拆分的视频提示词
- 每条镜头建议的 Flow 工作流
- 存帧 / 回灌 / 重做点
- Scenebuilder 拼接顺序
- 后期音频建议

## 关键提醒

- `Google Flow` 是平台，不等于模型栈
- 长片不要按“一条万能 prompt”去写
- 情绪高潮要从具体物件或具体动作长出来
- 如果是现实题材，先把职业、空间和时代感写死，再去谈风格化
