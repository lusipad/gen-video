# Flow 完整操作清单样例

这是一份面向 `Google Flow` 的完整执行模板。  
它不负责替你写故事，而是告诉你：当你已经拿到 `剧本 + 素材图提示词 + 镜头表` 之后，如何一步一步把项目在 Flow 里落地。

适用前提：

- 你已经有明确的故事和镜头结构
- 你准备在 `Google Flow` 中执行
- 你需要的是“操作顺序”和“检查点”，不是单条 prompt

## 0. 开始前确认

在进 Flow 之前，先确认这 6 件事：

1. 已锁定模型栈  
   例如：`Veo 3.1 + Nano Banana 2`

2. 已锁定项目比例  
   如果要大量使用 `Extend` 和 `Scenebuilder`，优先确认当前更稳的画幅。

3. 已有剧本或 beat 结构  
   至少要知道每个镜头的剧情作用。

4. 已有角色 / 场景 / 道具锚点  
   不然镜头连续性会非常差。

5. 已确认真实性约束  
   对真实地点、非遗、现实人物题材尤其重要。

6. 已决定哪些声音在 Flow 内处理，哪些留给后期  
   如果涉及儿童角色，关键对白优先留给后期。

## 1. 建立项目

1. 打开 `Google Flow`
2. 新建项目
3. 设置项目名
4. 确认当前项目使用的画幅
5. 先不要急着生成视频，先做素材资产

建议项目命名方式：

```text
[项目名]_[版本]_[日期]
```

例如：

```text
最后一班放映机_v1_2026-04-06
```

## 2. 先做素材图，不要直接做视频

先生成最小素材资产集：

1. 主角色参考图
2. 核心场景参考图
3. 核心道具参考图
4. 风格参考图
5. 如果有关键旧照片 / 历史影像感，也先做出来

最小建议资产表：

```text
C01 主角色
C02 次角色
S01 核心室内场景
S02 核心外景
P01 核心道具
ST01 风格参考
```

执行顺序建议：

1. 先生成角色图
2. 再生成场景图
3. 再生成道具图
4. 最后生成风格图

如果角色年龄、职业、地方感很重要，不要偷懒，先把角色图跑稳。

## 3. 给素材做“用途标记”

生成完图片后，不要直接继续跑视频。  
先在你自己的文档或命名里标清楚它们的用途。

建议标记方式：

```text
C01_老周_主参考
C02_林夏_主参考
S01_放映室_主场景
S02_露天放映空地_高潮场景
P01_16毫米放映机_主道具
ST01_银幕光束风格_氛围参考
```

至少要知道每张图是下面哪一类：

- 主参考
- 备用参考
- 风格参考
- 关键帧参考

## 4. 按镜头生成，不要按整片生成

在 Flow 中，长片要按镜头拆开做。  
不要试图用一条 prompt 直接生成 3 分钟内容。

建议把镜头分成三类：

1. `骨架镜头`
   必须讲剧情的主镜头

2. `插入镜头`
   用来补节奏、遮切点、拉时长的镜头

3. `重做镜头`
   需要存帧、回灌、提高质量的关键镜头

建议先做骨架镜头，再补插入镜头，最后重做关键镜头。

## 5. 选择合适的 Flow 工作流

每条镜头先判断该用哪一种：

### `Text to Video`

适合：

- 空镜
- 建立镜头
- 纯环境镜头
- 对连续性要求没那么高的过渡镜头

### `Ingredients to Video`

适合：

- 角色、场景、道具一致性要求高的镜头
- 有明确角色参考和场景参考的剧情镜头

### `Frames to Video`

适合：

- 情绪构图要稳的镜头
- 高潮关键镜头
- 需要固定起始画面时

### `Extend`

适合：

- 已经满意但节奏太短的镜头
- 需要多留呼吸感的镜头

不适合：

- 内容语义完全跳变的镜头
- 角色和空间已经不稳定的镜头

## 6. 第一轮只求“跑通”，别先追求完美

第一轮生成建议：

1. 每条镜头先出 `2 条`
2. 先选“最接近需求”的一条
3. 不要第一轮就在局部无限重刷
4. 先把整片骨架跑出来，再回头修

第一轮重点看：

- 人物对不对
- 地点对不对
- 道具结构对不对
- 情绪方向对不对
- 镜头是否能接上下一条

第一轮不要过度纠结：

- 极小的表情差异
- 次要背景小错误
- 音乐是否完美
- 不影响剧情的小瑕疵

## 7. 哪些镜头要立刻存帧

下面几类镜头，生成出满意版本后立刻 `Save frame as image`：

1. 角色第一次稳定出现的镜头
2. 主场景第一次稳定建立的镜头
3. 主道具第一次正确出现的镜头
4. 高潮前的关键情绪镜头
5. 高潮第一个成立的奇观镜头
6. 收尾动作最对的镜头

建议至少保留这些帧：

```text
F01 角色稳定帧
F02 场景稳定帧
F03 道具稳定帧
F04 高潮起点帧
F05 高潮扩散帧
F06 结尾收束帧
```

## 8. 第二轮只重做关键镜头

不要全片全部重做。  
第二轮只挑最重要的镜头重新生成或回灌。

优先重做这几类：

1. 情绪最强的镜头
2. 连续性最容易崩的镜头
3. 奇观第一次出现的镜头
4. 结尾收束镜头

常见重做方式：

1. 从第一轮里存下来的好帧出发
2. 用 `Frames to Video` 重新生成
3. 如果只是节奏不够，再用 `Extend`

## 9. 插入镜头怎么补

当整片太赶、太硬、太像“镜头清单”时，补插入镜头。

优先补这些：

1. 环境空镜
2. 手部特写
3. 道具特写
4. 静物特写
5. 人物呼吸和停顿
6. 灯光 / 火焰 / 水面 / 风的反应镜头

插入镜头的作用：

- 补情绪
- 遮剪辑切点
- 拉总时长
- 给观众呼吸

## 10. Scenebuilder 拼接顺序

当镜头基本确定后，再进 `Scenebuilder`。

建议顺序：

1. 先把骨架镜头按剧情顺序排好
2. 再插入过渡镜头
3. 最后插入情绪停顿镜头

拼接时重点检查：

- 镜头顺序是否清楚
- 情绪是否逐步递进
- 高潮前是否有足够铺垫
- 高潮后是否留出了回落空间
- 结尾是否停在正确的动作上

## 11. 音频处理建议

在 Flow 里优先处理：

- 环境音
- 动作音
- 氛围感
- 粗略音乐走向

留给后期处理：

- 关键对白
- 旁白
- 精准音乐起伏
- 哭点句子

如果题材涉及儿童角色，尤其建议：

- Flow 内保留环境和动作声音
- 核心台词后期再配

## 12. 最终检查表

导出前至少检查：

1. 角色年龄和气质前后一致
2. 场景没有从现实题材漂成泛化国风
3. 核心道具结构没有跑偏
4. 高潮从真实物件长出来，而不是凭空爆发
5. 收尾停在最该停的动作上
6. 关键泪点不依赖平台内生成对白
7. 全片节奏有起伏，不是一直平，也不是一直炸

## 13. 最简执行节奏

如果你只想记一个最短流程，记这 8 步：

1. 先锁模型和平台
2. 先做素材图
3. 再做骨架镜头
4. 第一轮先跑通全片
5. 满意镜头立刻存帧
6. 第二轮只重做关键镜头
7. 用插入镜头补情绪和节奏
8. 最后进 `Scenebuilder` 拼整片

- 剧本 / beat 文档
