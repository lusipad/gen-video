# Seedance 分镜输出示例

## 适用场景

- 用户明确要求 `Seedance` 风格时间轴分镜
- 重点是镜头分配、素材引用和声音设计
- 需要 `图片1 / 图片2 / 图片3` 这类显式素材引用格式

## 推荐调用方式

```text
使用 $gen-video，把这个故事改写成 Seedance 风格分镜输出。
```

## 用户输入示例

```text
使用 $gen-video，把这个故事改写成 Seedance 风格分镜：

一位守着老照相馆的摄影师即将关店。一个来做短期帮工的年轻女孩，帮他整理积满灰尘的旧底片、老相框和街坊合影。她想在关店前替他办一场“最后一夜橱窗展”，让这条街的人重新站回自己的照片里。整体要现实、克制、带一点旧城夜色和人情味，情绪慢慢递进，最后温暖收束。
```

## 预期输出

- 剧情结构
- 素材资产表
- `图片1 / 图片2 / 图片3` 引用方案
- 时间轴分镜
- 声音与配乐建议

## 关键提醒

- 只有在用户明确要 `Seedance` 时，才输出这种格式
- 时间轴分镜要覆盖完整片段时长
- `@素材` 引用必须和素材表一致
