# Examples

这个目录提供 `镜界.SKILL` 项目中 `$gen-video` 的最小可复制示例。  
目标不是展示成片，而是展示“什么样的任务描述，会触发什么样的交付形态”。

## 示例矩阵

| 示例 | 平台 / 模型 | 适合任务 | 主要输出 |
| --- | --- | --- | --- |
| [Flow_现实题材操作包.md](./Flow_现实题材操作包.md) | `Google Flow` | 现实题材长片或中短片 | 素材图、镜头表、回灌点、拼接顺序 |
| [Flow_完整操作清单样例.md](./Flow_完整操作清单样例.md) | `Google Flow` | 已有剧本和镜头表后的执行阶段 | 操作步骤、检查点、拼接与导出 |
| [Veo_NanoBanana_剧情短片提示词包.md](./Veo_NanoBanana_剧情短片提示词包.md) | `Veo 3.1 + Nano Banana 2` | 标准剧情短片 | 剧本、资产表、图片提示词、视频提示词 |
| [Seedance_分镜输出示例.md](./Seedance_分镜输出示例.md) | `Seedance` | 时间轴分镜项目 | 素材引用、分镜时间轴、声音设计 |

## 怎么用

把示例里的“用户输入示例”直接作为调用参考，根据你的题材替换故事内容、风格、时长和平台要求即可。

如果你不确定该用哪种示例：

- 要在 `Google Flow` 里落地，先看 `Flow_现实题材操作包`
- 已经有 prompt 包，只缺执行步骤，先看 `Flow_完整操作清单样例`
- 已经确定 `Veo 3.1 + Nano Banana 2`，先看 `Veo_NanoBanana_剧情短片提示词包`
- 明确要 `Seedance` 格式，先看 `Seedance_分镜输出示例`
