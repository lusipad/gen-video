# Veo + Nano Banana 剧情短片提示词包

- 用户已经明确模型栈为 `Veo 3.1 + Nano Banana 2`
- 目标主要是 `剧本 + 资产表 + 图片提示词 + 视频提示词`
- 不一定需要 `Google Flow` 执行包

## 推荐调用方式

```text
使用 $gen-video，把这个故事改写成 Veo 3.1 + Nano Banana 2 的剧本、素材清单、图片提示词和视频提示词。
```

## 用户输入示例

```text
使用 $gen-video，把这个现实题材短片改写成 Veo 3.1 + Nano Banana 2 的完整提示词包：

一个守着老钟表修理铺的老师傅，准备在月底关掉店门。一个总来换电池、修旧表的女孩发现，他修的不是钟表，而是整条街的时间记忆。她想在关店前帮他把修过的旧钟、旧表和那些顾客留下的故事，做成一支很短的记录片。整体风格要现实、克制、有人情味，前半段安静，后半段情绪逐步升起来，但不要走成热血动作片。
```

## 预期输出

- 故事结构
- 角色 / 场景 / 道具资产表
- Nano Banana 图片提示词
- Veo 3.1 视频提示词
- 关键镜头和结尾钩子

## 关键提醒

- 图片提示词不要直接拿去当视频提示词
- Veo 写过程，Nano Banana 写最终画面
- 如果是现实人物题材，先写真实性锚点和职业质感，再写情绪升级
