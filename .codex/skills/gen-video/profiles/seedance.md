# Seedance Profile

## 角色

`Seedance` 更偏向视频理解和自动分镜能力较强的路线。

## 原生强项

- 自动分镜
- 时间轴组织
- 参考图 / 参考视频驱动
- 连续镜头节奏感

## 核心判断

如果 Seedance 当前版本已经能原生稳定完成自动分镜，skill 不应再默认接管完整手工分镜。

## skill 应怎么介入

- `优先 native 或 hybrid`
- 把 Seedance 当成分镜主引擎
- skill 重点补：
  - 故事目标
  - 角色与地点锚点
  - 真实题材约束
  - 情绪弧线
  - 最终交付整理

## 什么时候切到 manual

- 只有当用户明确要求严格手工控制
- 或 Seedance 当前结果明显无法满足连续性 / 真实性 / 节奏要求

## 常见错误

- 模型已经擅长自动分镜，但 skill 仍强行输出一整套死板镜头表
- 把 Seedance 降格成“只吃 prompt 的普通视频模型”

## 高变动点

- 自动分镜能力边界
- 参考视频 / 参考图工作流
- 时间轴粒度
- 原生输出结构
