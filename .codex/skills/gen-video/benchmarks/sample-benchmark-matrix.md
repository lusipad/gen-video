# Sample Benchmark Matrix

| Case | 模型 / 平台 | 推荐模式 | 关注点 |
| --- | --- | --- | --- |
| 现实题材中片 | Flow | `hybrid` | 镜头编排、回灌、音频后期 |
| Seedance 自动分镜 | Seedance | `native` 或 `hybrid` | Skill 是否应退后 |
| 标准剧情短片 | Veo + Nano Banana | `hybrid` | 资产表和镜头分流 |
| 高控制广告片 | Veo + Nano Banana | `manual` | 严格镜头控制 |
| 非遗真实题材 | 任意 | `hybrid` | 真实性锚点优先 |
