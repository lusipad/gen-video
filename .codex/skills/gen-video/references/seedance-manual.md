# Seedance 完整参考手册

这份手册是面向 Codex 的执行版完整参考。目标不是介绍平台，而是帮助你在实际任务中快速判断：能做什么、不能做什么、该怎么问、该怎么写、最后该交付什么。

如果你时间有限，先读：

- [seedance-manual-summary.md](seedance-manual-summary.md)
- [seedance-playbook.md](seedance-playbook.md)

如果你需要完整模板、限制和示例，再读这份全文。

## 一、定位与能力范围

Seedance 适合把自然语言描述和多模态参考素材组合成可直接执行的短视频分镜提示词。

### 可处理的输入

- 文本描述
- 图片参考
- 视频参考
- 音频参考

### 适合的任务

- 叙事类短视频
- 角色动作展示
- 产品展示
- 空间漫游
- 口播视频
- 音乐卡点
- 视频续拍
- 视频编辑或剧情颠覆

### 核心思想

Seedance 最强的不是“单张图变动画”这一个动作，而是把：

- 画面风格
- 时间轴
- 声音设计
- 多模态引用

组合成一个统一的分镜提示词。

## 二、平台限制

### 素材数量限制

- 图片最多 9 张
- 视频最多 3 个
- 音频最多 3 个
- 总文件数最多 12 个

### 时长限制

- 视频参考总时长不超过 15 秒
- 音频参考总时长不超过 15 秒

### 注意事项

- 视频参考和音频参考通常更耗额度
- 暂不支持写实真人脸部素材
- 如果引用了素材，尽量明确写用途，不要只写一个 `@素材名`

## 三、推荐执行流程

### 第一步：确认需求是否适合 Seedance

当用户满足以下任一条件时，优先走 Seedance 路线：

- 明确说要 Seedance
- 明确说要时间轴分镜
- 有多张参考图、参考视频、参考音频，要统一组织到一个短片里
- 要求结果可直接复制到平台中使用

### 第二步：补齐最小必要信息

最少确认：

1. 你想讲什么
2. 要几秒
3. 比例是什么
4. 有没有图片、视频、音频参考

如果用户没有补全，也可以先按默认值起草第一版。

### 第三步：按四段结构起草

Seedance 提示词最稳的写法通常由四段组成：

1. 风格
2. 时间轴
3. 声音
4. 参考素材

### 第四步：做执行前检查

检查：

- 时间轴是否完整
- 镜头动作是否明确
- 声音是否写全
- `@素材` 是否标明用途
- 是否触发了平台限制

## 四、建议提问清单

如果用户没有给够信息，可以按下面顺序提问。

### 1. 故事与内容

- 这条视频主要讲什么
- 主角是谁
- 冲突或亮点是什么
- 是否有对白或旁白

### 2. 风格与比例

- 写实、动画、水墨、科幻、广告还是纪录片
- 竖屏、横屏还是电影宽屏
- 整体氛围偏温馨、悬疑、史诗还是紧张

### 3. 镜头与节奏

- 是要快剪、慢推、跟拍、环绕还是长镜头
- 是否需要音乐卡点
- 是否有强节奏动作

### 4. 素材情况

- 有没有角色参考图
- 有没有场景参考图
- 有没有运镜或动作参考视频
- 有没有音频、对白或配乐参考

## 五、标准提示词公式

推荐基础公式：

```markdown
【风格】[风格]，[时长]，[比例]，[氛围]

【时间轴】
0-X秒：[镜头] + [画面] + [动作] + [特效]
X-Y秒：[镜头] + [画面] + [动作] + [特效]
...

【声音】[配乐] + [音效] + [对白]

【参考】@图片1 [用途]，@视频1 [用途]，@音频1 [用途]
```

写时间轴时，每一段都至少交代：

1. 镜头怎么动
2. 画面里有什么
3. 主体在做什么
4. 这一段承担什么剧情功能

## 六、常用时间轴切法

### 15 秒叙事版

- 0-3 秒：建立场景
- 3-6 秒：引入主体或冲突
- 6-10 秒：高潮动作或情绪爆发
- 10-13 秒：转折或过渡
- 13-15 秒：结尾或落版

### 10-12 秒压缩版

- 0-2 秒：抓眼建立
- 2-5 秒：主体动作或信息展开
- 5-8 秒：高潮
- 8-10 或 12 秒：收尾

### 续拍版

- 第一段：说明延长
- 中间段：新增动作与发展
- 结尾段：与原视频无缝连接

## 十、多模态引用语法

推荐写法：

```markdown
@图片1 作为首帧
@图片2 作为尾帧
@图片3 作为角色形象参考
@图片4-6 作为场景参考
@视频1 参考运镜方式
@视频2 参考动作节奏
@音频1 用于配乐
@音频2 用于对白参考
```

原则：

- 每个 `@素材` 后面都尽量补用途
- 不要只引用，不说明其功能
- 同一素材在不同镜头中尽量保持同一用途

## 十一、特殊场景处理

### 1. 角色一致性

- 建议用户提供角色参考图
- 直接写 `@图片1 作为角色形象参考`
- 避免使用写实真人脸素材

### 2. 运镜或动作复刻

- 建议用户提供参考视频
- 可直接写 `完全参考@视频1的运镜和动作节奏`

### 3. 视频延长

- 直接写 `将@视频1延长X秒`
- 生成长度选择新增部分的长度，不是总长度

### 4. 剧情颠覆或编辑

- 先写保留
- 再写修改
- 再写新增

### 5. 音乐卡点

- 尽量提供节奏参考
- 时间轴要对齐音乐重拍

## 十二、示例提示词

### 示例一：情感叙事类

```markdown
电影级写实风格，15秒，2.35:1电影宽屏，温馨家庭氛围

0-3秒：中景跟随镜头，男人疲惫地走在走廊，脚步逐渐变缓，最后停在家门口
3-5秒：脸部特写，男人深呼吸，调整情绪，收起负面情绪，表情变得轻松
5-7秒：特写手部动作，翻找出钥匙，插入门锁，门打开
7-12秒：室内中景，小女儿和宠物狗欢快地跑过来迎接，男人蹲下拥抱他们
12-15秒：近景，男人脸上洋溢着幸福的笑容，室内暖光营造温馨氛围

【声音】轻缓钢琴配乐，脚步声、开门声、孩子笑声
【参考】@图片1 男人形象，@图片2 女儿形象，@图片3 宠物狗形象
```

### 示例二：动作打斗类

```markdown
中国水墨武侠风格，15秒，16:9，枫叶飘落的秋季场景

0-2秒：远景，两位侠客对峙
2-4秒：快速推近，两人眼神交锋，杀气弥漫
4-9秒：中景快速剪辑，长枪突刺，双刀格挡，武器碰撞，火花四溅，模仿@视频1的动作节奏
```

## 十四、优质提示词检查表

高质量 Seedance 提示词通常满足：

- 时间轴清晰
- 镜头语言明确
- 动作描述具体
- `@素材` 用途规范
